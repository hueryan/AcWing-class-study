1. 安装`django_redis`

   `pip install django_redis`

2.  配置`settings.py`

   ```py
   CACHES = {
       'default': {
           'BACKEND': 'django_redis.cache.RedisCache',
           'LOCATION': 'redis://127.0.0.1:6379/1',
           "OPTIONS": {
               "CLIENT_CLASS": "django_redis.client.DefaultClient",
           },
       },
   }
   USER_AGENTS_CACHE = 'default'
   ```

3. 启动`redis-server`

   `sudo redis-server /etc/redis/redis.conf`

**Django 操作redis**

```py
python3 ~/app/manege.py shell

from django.core.cashe import cache
cache.keys('*')  # 查找所有关键字
cache.set('yqy', 1, 5)  # key, value, timeout(s)
cache.set('yqy', 1, None)  # 不会过期
cache.has_key('yqy')  # 查看 'yqy' 这个key存在不
cache.get('yqy')  # 查询值结果
cache.delete('yqy')
```

**清空**

```py
from django.core.cashe import cache

for key in cache.keys('*'):
    cache.delete(key)
```
