### 友情提示

**启动服务顺序**

0. 启动nginx服务`sudo /etc/init.d/nginx start` 

1. 启动redis-server服务 `sudo redis-server /etc/redis/redis.conf` 

2. 启动uwsgi服务 `uwsgi --ini ~/app/scripts/uwsgi.ini`   # 启动https server

3. 启动 django_channels服务 `daphne -b 0.0.0.0 -p 5015 ~/app/app.asgi:application`  # 启动wss server
