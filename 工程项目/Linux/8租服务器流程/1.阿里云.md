# [阿里云](https://www.aliyun.com/)

服务器-包年包月 带宽-按流量付费

1. 登录之后点击控制台跳转[页面](https://home.console.aliyun.com/home/dashboard/ProductAndService) 左侧菜单 -计算-云服务器ECS跳转[页面](https://ecs.console.aliyun.com/home) 点击左侧实例 进入-创建实例 
2. 点击 自定义购买 
   1. 包年包月-选最近地区
   2. `1vCPU 2GiB`  镜像 `Ubantu` 
3. 网络与安全组
   1. 选用按使用流量
   2. 峰值拉满
   3. 新建安全组-控制端口开放
4. 管理设置
   1. 自定义密码登录
5. 云服务器文档 点击控制台-云服务器ECS-概览-右侧帮助与文档 [url](https://help.aliyun.com/zh/ecs/)
