# 目录

## 简略

`bin` 常用的可执行文件命令 `ls` 

`lib` 库文件、静态类库 `C++头文件` 

`etc` 配置文件 `nginx配置文件` 

`var` 网站日志 `log` 

`home` 用户家目录 

`proc` 进程和内核信息 `cpuinfo` 

`root` 跟用户目录

## 📁 **核心系统目录**

### `/`

- **根目录**，所有目录的起点

### `/bin` (Binary)

- **基础命令**：`ls`, `cp`, `rm`等
- 系统启动和恢复所需的基本命令
- 所有用户可执行

### `/sbin` (System Binary)

- **系统管理命令**：`fdisk`, `ifconfig`, `init`
- 需要root权限执行
- 系统维护和修复工具

### `/boot`

- **启动文件**：内核、引导加载程序(GRUB)
- 包含：`vmlinuz`(内核)、`initramfs`、`grub/`

### `/dev` (Device)

- **设备文件**：硬件设备的抽象
- 如：`/dev/sda`(磁盘)、`/dev/tty`(终端)、`/dev/null`

### `/etc` (Etcetera)

- **配置文件**：系统和应用程序配置
- 包含：`/etc/passwd`(用户)、`/etc/fstab`(挂载)、`/etc/network/`

### `/lib` & `/lib64`

- **共享库**：`/bin`和`/sbin`程序的依赖库
- 内核模块：`/lib/modules/`

## 🏠 **用户相关目录**

### `/home`

- **用户家目录**：每个用户在`/home/用户名/`下有自己的空间
- 存储个人文件、配置（`~/.bashrc`）、下载等

### `/root`

- **管理员家目录**：root用户的家目录

## 🛠️ **运行和临时目录**

### `/proc` (Process)

- **虚拟文件系统**：进程和内核信息
- 实时信息：`/proc/cpuinfo`、`/proc/meminfo`
- 进程目录：`/proc/[PID]/`

### `/sys`

- **系统信息**：设备、驱动、内核参数
- 统一设备模型接口

### `/tmp`

- **临时文件**：所有用户可读写
- 重启可能清空（取决于配置）

### `/run`

- **运行时数据**：系统启动后的临时文件
- 如：PID文件、套接字、锁文件

## 📦 **程序和资源目录**

### `/usr` (User System Resources)

- **用户程序资源**：类似Windows的`Program Files`
- 子目录：
  - `/usr/bin/`：用户命令
  - `/usr/lib/`：程序库
  - `/usr/share/`：共享数据（文档、图标等）
  - `/usr/local/`：本地安装的软件
  - `/usr/include/`：C/C++头文件

### `/opt` (Optional)

- **可选软件包**：第三方大型应用
- 每个软件在独立子目录中

## 📊 **可变数据目录**

### `/var` (Variable)

- **可变数据**：经常变化的文件
- 重要子目录：
  - `/var/log/`：**日志文件**（`syslog`、`auth.log`等）
  - `/var/cache/`：应用程序缓存
  - `/var/spool/`：队列数据（邮件、打印）
  - `/var/www/`：Web服务器文件（Apache/Nginx）
  - `/var/lib/`：数据库和状态信息

## 💾 **挂载点目录**

### `/mnt` (Mount)

- **临时挂载点**：手动挂载设备
- 如：`sudo mount /dev/sdb1 /mnt/usb`

### `/media`

- **自动挂载点**：可移动媒体自动挂载
- 如：U盘、光盘

## 🔧 **特殊用途目录**

### `/srv` (Service)

- **服务数据**：网站、FTP等服务器数据
- 如：`/srv/www/`、`/srv/ftp/`

### `/lost+found`

- **恢复文件**：`fsck`修复的文件碎片

## 💡 **关键要点**

1. **系统配置** → `/etc/`
2. **用户文件** → `/home/用户名/`
3. **程序文件** → `/usr/` 或 `/opt/`
4. **日志文件** → `/var/log/`
5. **临时文件** → `/tmp/` 或 `/var/tmp/`





1. **/bin**：存放系统的基本命令，这些命令在单用户模式下也可以使用，所有用户都可以执行。
2. **/boot**：包含启动Linux系统所需的文件，如内核、引导加载程序（grub）等。
3. **/dev**：设备文件目录，Linux将硬件设备抽象为文件，存放在此。
4. **/etc**：系统配置文件存放目录，如网络配置、用户账户、系统服务配置等。
5. **/home**：普通用户的家目录，每个用户通常在此有一个以用户名命名的子目录，用于存放个人文件。
6. **/lib**和**/lib64**：存放系统最基本的动态链接库，32位系统是/lib，64位系统还有/lib64。
7. **/media**：可移动媒体（如U盘、光盘）的挂载点。
8. **/mnt**：临时挂载文件系统的目录，常用于挂载硬盘、网络共享等。
9. **/opt**：可选应用程序的安装目录，通常用于安装第三方软件。
10. **/proc**：虚拟文件系统，提供内核和进程信息的接口，这些文件不在硬盘上，而是在内存中。
11. **/root**：系统管理员（root）的家目录。
12. **/sbin**：存放系统管理命令，通常需要root权限才能执行。
13. **/srv**：存放服务相关数据，如Web服务器的网站文件。
14. **/sys**：虚拟文件系统，提供内核设备、驱动程序等信息的接口。
15. **/tmp**：临时文件目录，所有用户都可以读写，系统重启时可能会清空。
16. **/usr**：用户应用程序和文件存放的目录，类似于Windows的Program Files。
    - /usr/bin：用户安装的应用程序的可执行文件。
    - /usr/lib：应用程序的库文件。
    - /usr/local：用户手动安装的软件，通常由源码编译安装。
    - /usr/share：应用程序的共享资源，如文档、图标等。
17. **/var**：存放经常变化的文件，如日志、缓存、邮件等。
    - /var/log：系统日志文件。
    - /var/cache：应用程序缓存数据。
    - /var/spool：队列数据，如打印队列、邮件队列。
