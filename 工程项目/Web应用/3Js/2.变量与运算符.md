`let`与`const`
用来声明变量，作用范围为当前作用域。

- `let`用来定义变量；
- `const`用来定义常量；

```js
let s = "link", x = 5;

let d = {
    name: "hu",
    age: 18,
}

const n = 100;
```

---

**变量类型**

- `number`：数值变量，例如1, 2.5
- `string`：字符串，单引号与双引号均可。字符串中的每个字符为只读类型。
- `boolean`：布尔值
- `object`：对象，类似于C++中的指针，例如`[1, 2, 3]`，{name: "yxc", age: 18}，null
- `undefined`：未定义的变量

类似于Python，JavaScript中的变量类型可以动态变化。

---

**运算符**
与C++、Python、Java类似，不同点：

- `**`表示乘方
- 等于与不等于用`===`和`!==`







- 获取x的类型：`console.log(typeof x)` 

- 修改字符串

  ```js
  let s1 = "abcdefg"
  let s2 = s1.substr(0, 1) + 'x' + s1.substr(2)
  ```

- 字典引用

  ```js
  let d = {
      name: 'abc',
      age: 18,
  }
  console.log(d['name'], d.age)
  d.school = 'asd'
  delete d.schook
  ```

- 取整数

  ```js
  console.log(parseInt(5/3))
  ```
